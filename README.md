# GhostForensics Scam Analyzer

A free, educational risk analysis tool designed to help everyday people 
identify scams, phishing attempts, fake job offers, and suspicious messages 
before they become victims.

## What it does

- Analyzes suspicious messages, emails, URLs, and domains
- Detects 10+ scam categories including phishing, romance scams, 
  government impersonation, fake tech support, and more
- Provides a risk score from 0 to 100
- Explains red flags in simple, clear language
- Suggests safe next steps for each situation

## Scam categories detected

- Job Scam
- Phishing
- Marketplace Scam
- Investment Scam / Pig Butchering
- Romance Scam
- Tech Support Scam
- Government Impersonation
- Lottery and Advance Fee Scam
- Rental Scam
- Delivery and Smishing Scam

## How to use

1. Clone or download this repository
2. Open `index.html` with Live Server in VS Code
3. Paste a suspicious message, email, or domain
4. Select the case type
5. Click Analyze Risk

## Tech stack

- HTML, CSS, JavaScript
- Modular JSON knowledge base
- No external APIs required for core analysis
- Mobile friendly
